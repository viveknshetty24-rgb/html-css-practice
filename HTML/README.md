# HTML Practice

This folder contains my **HTML practice files** created while learning the basics of web development.

## About

The purpose of this folder is to practice and understand **HTML structure and fundamental concepts** by building simple web pages.

## Topics Covered

* Basic HTML structure
* Headings and paragraphs(h1-h6,p)
* Lists and tables(li,ui,dl,table)
* Links and images (href,img)
* Audio and video elements (mp3,mp4)
* Semantic Tags
* Inline-Block elements
